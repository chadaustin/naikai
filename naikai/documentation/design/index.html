Naikai

Introduction

I ran into several problems during the creation of the Sphere RPG engine. Naikai is an attempt to create a new engine to remedy those problems and provide a basic framework for any games that I wish to make myself.

Goals

  • Portable
  • Efficient
  • Highly componentized
  • Flexible (i.e. easily maintainable)
  • Internationalizable

Non-Goals

  • Easy for non-developers
  • Specific domain solutions

Components

  • XPCOM
  • Windowing System
  • Graphics
  • Audio
  • Input
